#####必读事项
一、刷入协议：
1.自行承担搞机刷机的常见风险：卡米，黑砖等。
2自行承担.刷入本模块风险：烫手，硬件损耗加快，烧主板，电池爆废。
3.请勿在高温环境充电，游戏，或者其他高功耗发热使用。
4.不删温控，卸载恢复官方。执行uninstall.sh卸载更完整干净。

二、作者提供的服务：
1.在空闲时间里，无偿帮助同作者机型用户刷机，解答问题。
2.接受所有的反馈问题，积极解决问题。更或者接受所以指正和建议。


#####使用方法
一、实现原理:
1.用空温控文件代替温控文件。实现删温控，满血快充。以及停用MIUI的“analytics，joyose，和电量和性能云控服务组件”且清理云控缓存。实现桌面，应用，游戏等场景不再掉帧。

二、食用方法：
1.Magisk面具24.0版本以上无脑刷入即可。
2.目录system是映射到系统的空温控文件目录。
3.目录thermal是自定义的云控文件目录。
4.目录root是定时执行I_Control.sh的任务，利用的linux的crontab服务。
5.I_Control.sh是智障控制淦云控，更新module.prop的脚本。
6.uninstall.sh卸载模块脚本，执行它卸载脚本更干净。
7.Reset.sh用于重置智障淦云控的时间集，如有掉网络(WLAN，数据网络，GPS，NFC，热点)问题执行这个脚本就行。
8.更新模块面板.sh用于跳转本模块永久更新链接的脚本。
省流：1.无脑面具刷入安装。2.面具app里卸载/root执行uninstall.sh卸载(更干净)。3.有其他问题再看其他的。
